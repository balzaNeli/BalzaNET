## 🎨 Sobre o Projeto

Este projeto foi desenvolvido focado na experiência do usuário (UX) e em uma identidade visual forte (Dark Mode com detalhes em Dourado). A BalzaNET simula um provedor de internet de alta performance, contando com:

* **Landing Page:** Apresentação de planos, diferenciais e CTAs.
* **Design System:** Base de tokens (cores, fontes e espaçamentos) para consistência visual.
* **Área do Cliente:** Dashboard exclusivo para gerenciamento de faturas e consumo.
* **Suporte Técnico:** Integração com PHP e MySQL para abertura de chamados.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza uma stack equilibrada entre design moderno e funcionalidade:

- [x] **HTML5** & **CSS3** (Variáveis CSS, Flexbox, Grid)
- [x] **Google Fonts** (Montserrat & Nunito)

## 📱 Responsividade

A interface foi projetada com foco em **Mobile-First**. A barra de navegação se adapta para um menu hamburger no mobile e a Área do Cliente utiliza uma navegação horizontal deslizante para garantir a melhor experiência em qualquer dispositivo.
